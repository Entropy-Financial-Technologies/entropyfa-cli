# Documentation

The docs in this folder are intentionally split by audience.

## User Docs

- [embedded-data.md](embedded-data.md) — reference for `entropyfa data lookup`, including supported keys, required params, and example responses from the current embedded dataset
- [compute-visuals.md](compute-visuals.md) — user-facing guide to terminal dashboards and chart-like compute output, including the Monte Carlo projection visual
- [openclaw.md](openclaw.md) — install and usage guide for the official OpenClaw skill, including `clawhub install entropyfa` and recommended agent workflows

## Maintainer Docs

- [data-pipeline.md](data-pipeline.md) — contributor and maintainer guide for the data verification pipeline, including `prepare`, `run-agents`, `review`, `apply`, `snapshot`, and `validate`
